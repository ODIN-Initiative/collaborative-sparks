# collaborative-sparks

Collaborative Sparks has a separate Github account - https://github.com/CollaborativeSparks.  And, a temporary Website - https://collaborativesparks.github.io/CollaborativeSparksWebsiteContent/ 

Collaborative Sparks is working to spread insight about the use of
social governance habits for decentralized organizations to be
impactful without the traditional methods of organizational hierarchy.
